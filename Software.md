# 🛠️ VLSI / Semiconductor Toolchain (Phase-wise)

> 🎯 Goal: Learn and use the RIGHT tools for each phase  
> ⚠️ Focus: Essential tools only (Industry + Open Source)

---

# 📦 PHASE 1 — FOUNDATION TOOLS

## 🔹 RTL Simulation & Debugging (CORE)
- Icarus Verilog (iverilog) — Verilog compiler
- ModelSim / QuestaSim — Industry simulator (Paid)
- GTKWave — Waveform viewer ⭐ (VERY IMPORTANT)

## 🔹 Code Editor / IDE
- VS Code (with extensions):
  - Verilog HDL
  - C/C++
  - Python

## 🔹 Linux Environment
- WSL2 (Windows) / Native Linux (Ubuntu)
- Bash terminal

## 🔹 Version Control
- Git
- GitHub

## 🔹 Programming
- GCC (C compiler)
- Python 3

---

# 🚀 PHASE 2 — VLSI CORE TOOLS

## 🔹 Advanced Simulation
- ModelSim / QuestaSim (Paid)
- Verilator (Lint + fast simulation) ⭐

## 🔹 SystemVerilog / Verification
- QuestaSim (preferred) (Paid)
- VCS (Synopsys) (Paid)

## 🔹 Synthesis (RTL → Gate)
- Yosys (Open-source) ⭐
- Synopsys Design Compiler (Paid)

## 🔹 Static Timing Analysis (STA)
- OpenSTA (Open-source)
- Synopsys PrimeTime (Paid)

## 🔹 Physical Design (PnR)
- OpenROAD ⭐ (Open-source full flow)
- Cadence Innovus (Paid)
- Synopsys ICC2 (Paid)

## 🔹 Layout & Verification
- Magic VLSI (Layout)
- Netgen (LVS)
- KLayout (GDS viewer)
